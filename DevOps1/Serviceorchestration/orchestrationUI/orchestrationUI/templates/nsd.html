NSD DESCRIPTION



  • The Network Service Descriptor (NSD) consists of static information regarding nsd base element,Connection Point,Service deployment flavour
  • It is used by the NFV
    • Orchestrator to instantiate a Network Service, which would be formed by one or more VNF Forwarding Graphs, VNFs, PNFs and VLs.
  • The NSD also describes deployment flavours of Network Service.
  • Nsd is Network service descriptor
